# inet_pfc
INET simulation of PFC

## How to Use
- Download and Install Omnet++ 5.6 & INET 4.2
- Create a New Project in Omnetpp IDE
- Import the Source Files
- Add Project Dependency (Properties->Project References)
- Build and Run
